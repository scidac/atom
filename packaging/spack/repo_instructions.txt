This is an AToM spack repo.

It can be added to your spack repo list with

spack repo add /path/to/this/directory/scidac-atom

This will allow you to do 

spack install <package>

For any of the AToM packages in this repo.

After installing, you may need to set up your environment with "spack load <package"

Note: On MacOS, "spack load" requires having the environment-module spack package installed.
